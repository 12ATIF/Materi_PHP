üì¶ Bagian 1: Apa itu Fungsi?

ü§î Analogi Sederhana

Fungsi itu seperti mesin khusus. Kamu masukkan bahan (parameter), mesin memproses, lalu keluar hasil (return value).

üéØ Mengapa Pakai Fungsi?

‚ùå Tanpa Fungsi (Kode Berulang)
// Menghitung luas berkali-kali
$sisi1 = 5;
$luas1 = $sisi1 * $sisi1;

‚úÖ Dengan Fungsi (Efisien)

    return $sisi * $sisi;

üíª Bagian 2: Membuat Fungsi

fungsi-dasar.php 
<?php
// Fungsi TANPA parameter
function sapa() {
    echo "Halo, selamat datang!";
}

// Memanggil fungsi
sapa();  // Output: Halo, selamat datang!
sapa();  // Bisa dipanggil berulang kali

// Fungsi DENGAN parameter
function sapaUser($nama) {
    echo "Halo, $nama! Selamat belajar PHP.";
}

sapaUser("Budi");  // Output: Halo, Budi! Selamat belajar PHP.
sapaUser("Ani");   // Output: Halo, Ani! Selamat belajar PHP.
?>

üìñ Penjelasan Struktur Fungsi:

function namaFungsi($parameter) { // kode }
Bagian Penjelasan
function Kata kunci untuk membuat fungsi
namaFungsi Nama yang kamu pilih (huruf, angka, underscore)
$parameter Data yang dikirim ke fungsi (opsional)
{ ... } Blok kode yang akan dijalankan

üì• Bagian 3: Parameter & Argument

parameter.php 
<?php
// BANYAK PARAMETER
function perkenalan($nama, $kelas, $jurusan) {
    echo "Nama: $nama <br>";
    echo "Kelas: $kelas <br>";
    echo "Jurusan: $jurusan <br>";
}

perkenalan("Budi", "XI", "RPL");
perkenalan("Ani", "XII", "TKJ");

// PARAMETER DEFAULT VALUE
function hitungDiskon($harga, $diskon = 10) {
    // Jika diskon tidak diisi, nilainya 10
    $potongan = $harga * ($diskon / 100);
    return $harga - $potongan;
}

echo hitungDiskon(100000);      // 90000 (diskon 10%)
echo hitungDiskon(100000, 25);  // 75000 (diskon 25%)
?>
üí°
Parameter vs Argument:
  • ‚Ä¢ Parameter = variabel di definisi fungsi ($nama, $kelas)
  • ‚Ä¢ Argument = nilai yang dikirim saat memanggil ("Budi", "XI")

üì§ Bagian 4: Return Value

üîÑ Echo vs Return

echo

Langsung tampilkan ke layar

Tidak bisa disimpan ke variabel

return

Kembalikan nilai ke pemanggil

Bisa disimpan ke variabel

return.php 
<?php
// Fungsi dengan RETURN
function tambah($a, $b) {
    return $a + $b;  // Kembalikan hasil
}

$hasil = tambah(5, 3);   // $hasil = 8
echo "Hasil: $hasil";     // Output: Hasil: 8

// Contoh praktis: Hitung luas berbagai bangun
function luasPersegi($sisi) {
    return $sisi * $sisi;
}

function luasSegitiga($alas, $tinggi) {
    return 0.5 * $alas * $tinggi;
}

function luasLingkaran($jariJari) {
    return 3.14 * $jariJari * $jariJari;
}

// Pakai hasilnya
$lPersegi = luasPersegi(5);        // 25
$lSegitiga = luasSegitiga(10, 8);   // 40
$lLingkaran = luasLingkaran(7);     // 153.86

echo "Persegi: $lPersegi <br>";
echo "Segitiga: $lSegitiga <br>";
echo "Lingkaran: $lLingkaran <br>";
?>
‚ö†Ô∏è
Penting! Setelah return, kode di bawahnya tidak akan dijalankan. Return langsung menghentikan fungsi.

üõ†Ô∏è Bagian 5: Fungsi Bawaan PHP

PHP punya ratusan fungsi bawaan yang siap pakai. Berikut yang paling sering digunakan:

üìù Fungsi String

Fungsi Kegunaan Contoh Hasil
strlen() Panjang string strlen("Halo") 4
strtoupper() Huruf besar semua strtoupper("halo") HALO
strtolower() Huruf kecil semua strtolower("HALO") halo
ucfirst() Kapital huruf pertama ucfirst("budi") Budi
ucwords() Kapital setiap kata ucwords("budi santoso") Budi Santoso
trim() Hapus spasi di ujung trim(" halo ") halo
str_replace() Ganti teks str_replace("a","o","baca") boco

üî¢ Fungsi Matematika

Fungsi Kegunaan Contoh Hasil
abs() Nilai absolut abs(-5) 5
round() Pembulatan round(3.7) 4
ceil() Pembulatan ke atas ceil(3.1) 4
floor() Pembulatan ke bawah floor(3.9) 3
rand() Angka acak rand(1, 100) 1-100
max() Nilai terbesar max(5, 10, 3) 10
min() Nilai terkecil min(5, 10, 3) 3

üìÖ Fungsi Tanggal

tanggal.php 
<?php
// Fungsi date() dengan format
echo date("d/m/Y");         // 30/01/2026
echo date("d F Y");         // 30 January 2026
echo date("l, d F Y");      // Friday, 30 January 2026
echo date("H:i:s");         // 18:30:45

// Format yang sering dipakai
// d = tanggal (01-31)
// m = bulan (01-12)
// Y = tahun 4 digit
// H = jam 24-jam
// i = menit
// s = detik
// l = nama hari (Monday, Tuesday, ...)
// F = nama bulan (January, February, ...)
?>
