üìñ Apa itu Fungsi?

üí°
Bayangkan begini: Fungsi seperti mesin khusus. Kamu masukkan bahan (parameter), mesin akan memproses, lalu keluar hasil (return).

Fungsi adalah blok kode yang bisa dipanggil berkali-kali. Dengan fungsi, kode jadi lebih rapi dan tidak perlu ditulis ulang.

Fungsi Sederhana

fungsi.php 
<?php
// Membuat fungsi
function sapa() {
    echo "Halo, selamat datang!";
}

// Memanggil fungsi
sapa();  // Output: Halo, selamat datang!
sapa();  // Bisa dipanggil berkali-kali
?>

Fungsi dengan Parameter

parameter.php 
<?php
// Fungsi dengan 1 parameter
function sapa($nama) {
    echo "Halo, $nama!";
}

sapa("Budi");   // Output: Halo, Budi!
sapa("Ani");    // Output: Halo, Ani!

// Fungsi dengan banyak parameter
function perkenalan($nama, $umur, $kelas) {
    echo "Nama: $nama, Umur: $umur, Kelas: $kelas";
}

perkenalan("Budi", 16, "X RPL 1");
?>

Fungsi dengan Return

return.php 
<?php
// Fungsi yang mengembalikan nilai
function tambah($a, $b) {
    return $a + $b;
}

$hasil = tambah(5, 3);
echo "Hasil: $hasil";  // Output: Hasil: 8

// Fungsi hitung luas persegi
function luasPersegi($sisi) {
    return $sisi * $sisi;
}

$luas = luasPersegi(4);
echo "Luas: $luas";  // Output: Luas: 16
?>
üß†
Tips: echo langsung menampilkan ke layar. return mengembalikan nilai yang bisa disimpan ke variabel.

Fungsi Bawaan PHP

PHP punya banyak fungsi bawaan yang bisa langsung dipakai:

built-in.php 
<?php
// String functions
echo strlen("Hello");        // 5 (panjang string)
echo strtoupper("hello");    // HELLO
echo strtolower("HELLO");    // hello

// Math functions
echo abs(-5);     // 5 (nilai absolut)
echo round(3.7);  // 4 (pembulatan)
echo rand(1, 10); // Angka random 1-10

// Date functions
echo date("d/m/Y");  // Tanggal hari ini
echo date("H:i:s");  // Jam sekarang
?>
