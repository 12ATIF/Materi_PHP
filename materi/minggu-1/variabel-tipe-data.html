Bayangkan begini: Variabel itu seperti kotak penyimpanan yang punya nama. Kamu bisa menyimpan data apapun ke dalam kotak itu.

Di PHP, variabel diawali dengan tanda $ (dollar).

variabel.php 
<?php
// Membuat variabel
$nama = "Budi";           // Menyimpan teks
$umur = 17;               // Menyimpan angka
$tinggi = 165.5;          // Menyimpan angka desimal
$sudahLulus = false;      // Menyimpan true/false

// Menampilkan variabel
echo "Nama: " . $nama;    // Output: Nama: Budi
echo "Umur: " . $umur;    // Output: Umur: 17
?>
