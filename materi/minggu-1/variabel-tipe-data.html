ğŸ“š Bagian 3: Array (Kumpulan Data)

Array adalah variabel yang bisa menyimpan banyak nilai sekaligus. Bayangkan seperti lemari dengan banyak laci bernomor.

array.php 
<?php
// ARRAY INDEXED - index dimulai dari 0
$buah = ["Apel", "Jeruk", "Mangga"];
//         [0]      [1]      [2]

echo $buah[0];  // Output: Apel
echo $buah[1];  // Output: Jeruk
echo $buah[2];  // Output: Mangga

// ARRAY ASSOCIATIVE - index berupa nama (key)
$siswa = [
    "nama" => "Budi",
    "kelas" => "XI RPL",
    "umur" => 17
];

echo $siswa["nama"];   // Output: Budi
echo $siswa["kelas"];  // Output: XI RPL

// Menambah data ke array
$buah[] = "Durian";        // Tambah di akhir
$siswa["hobi"] = "Coding"; // Tambah dengan key

// Menghitung jumlah elemen
echo count($buah);  // Output: 4
?>

ğŸ“– Perbedaan Array Indexed vs Associative:

Indexed Array

Index otomatis berupa angka (0, 1, 2, ...)

$buah[0], $buah[1]
Associative Array

Index berupa nama/key yang kamu tentukan

$siswa["nama"], $siswa["kelas"]

<?php

// Cara 2: Pakai kutip ganda
$salam = "Halo, nama saya $depan $belakang";
echo $salam;    // Output: Halo, nama saya Budi Santoso

// Gabung angka dan teks
$umur = 17;
echo "Umur saya " . $umur . " tahun";
?>
