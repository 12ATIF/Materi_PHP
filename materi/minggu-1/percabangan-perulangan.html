ğŸ¤” Kapan Pakai While?

Gunakan while ketika perulangan bergantung pada kondisi dan kamu tidak tahu pasti berapa kali akan diulang.

    echo "Hitungan: $i <br>";
    $i++;  // WAJIB! Tanpa ini = infinite loop
}

while ($detik > 0) {
    echo "$detik... ";
    $detik--;
}
echo "Waktu habis! â°";

    echo "Angka: $angka <br>";  // Akan dijalankan minimal 1x

?>
while

Cek kondisi sebelum menjalankan

Bisa 0 kali jika kondisi awal false

do-while

Cek kondisi setelah menjalankan

Minimal 1 kali pasti dijalankan
