ğŸ’¡
Bayangkan begini: Perulangan seperti mesin fotokopi. Kamu tentukan berapa kali mau difotokopi, mesin akan mengulang tugas itu.

For Loop

for-loop.php 
<?php
// Menampilkan angka 1 sampai 5
for ($i = 1; $i <= 5; $i++) {
    echo "Angka ke-$i <br>";
}

// Output:
// Angka ke-1
// Angka ke-2
// Angka ke-3
// Angka ke-4
// Angka ke-5
?>

While Loop

while-loop.php 
<?php
$i = 1;
while ($i <= 3) {
    echo "Hitungan: $i <br>";
    $i++;  // Jangan lupa tambah nilai!
}
?>

Foreach (untuk Array)

foreach.php 
<?php
$buah = ["Apel", "Mangga", "Jeruk"];

foreach ($buah as $item) {
    echo "Buah: $item <br>";
}

// Output:
// Buah: Apel
// Buah: Mangga
// Buah: Jeruk
?>
