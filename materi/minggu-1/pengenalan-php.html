Install PHP dengan Laragon

Untuk menjalankan PHP, kamu butuh web server. Cara paling mudah adalah pakai Laragon (Windows) atau XAMPP.

  1. Download Laragon dari laragon.org
  2. Install dan jalankan Laragon
  3. Klik "Start All" untuk menyalakan Apache & MySQL
  4. Buka folder C:\laragon\www untuk menyimpan file PHP

// Bisa juga pakai tanda kutip tunggal
echo 'Selamat datang di dunia PHP';
?>
