üìñ Apa itu Form?

üí°
Bayangkan begini: Form seperti formulir pendaftaran. User mengisi data, klik tombol submit, lalu data dikirim ke server untuk diproses.

Form HTML Dasar

form.html 
<form action="proses.php" method="POST">
    <label>Nama:</label>
    <input type="text" name="nama">
    
    <label>Email:</label>
    <input type="email" name="email">
    
    <label>Pesan:</label>
    <textarea name="pesan"></textarea>
    
    <button type="submit">Kirim</button>
</form>

Perbedaan GET dan POST

GET

  • ‚Ä¢ Data muncul di URL
  • ‚Ä¢ Cocok untuk pencarian
  • ‚Ä¢ Ada batasan panjang data
  • ‚Ä¢ Tidak aman untuk password

POST

  • ‚Ä¢ Data tersembunyi di body
  • ‚Ä¢ Cocok untuk form login
  • ‚Ä¢ Tidak ada batasan data
  • ‚Ä¢ Lebih aman untuk password

Mengakses Data Form di PHP

proses.php 
<?php
// Mengakses data dari form POST
$nama = $_POST['nama'];
$email = $_POST['email'];
$pesan = $_POST['pesan'];

echo "Halo, $nama!";
echo "Email kamu: $email";
echo "Pesan: $pesan";

// Untuk method GET, gunakan $_GET
// $cari = $_GET['keyword'];
?>
