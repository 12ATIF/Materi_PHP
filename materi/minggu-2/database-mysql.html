üìñ Apa itu Database?

üí°
Bayangkan begini: Database seperti lemari arsip digital. Data disimpan rapi dalam tabel-tabel, mudah dicari dan dikelola.

Membuat Database di phpMyAdmin

  1. Buka browser, ketik localhost/phpmyadmin
  2. Klik "Baru" atau "New" di sidebar kiri
  3. Ketik nama database (misal: belajar_php)
  4. Klik "Buat" atau "Create"

Membuat Tabel

SQL - Buat Tabel 
CREATE TABLE siswa (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100) NOT NULL,
    kelas VARCHAR(20),
    email VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

üîå Koneksi PHP ke MySQL

koneksi.php 
<?php
// Data koneksi
$host = "localhost";
$user = "root";        // Default Laragon
$pass = "";            // Kosong di Laragon
$db   = "belajar_php";

// Membuat koneksi
$conn = mysqli_connect($host, $user, $pass, $db);

// Cek koneksi
if (!$conn) {
    die("Koneksi gagal: " . mysqli_connect_error());
}

echo "Koneksi berhasil! ‚úì";
?>
üß†
Tips: Simpan file koneksi terpisah (koneksi.php), lalu panggil dengan include di file lain.

Menggunakan Koneksi di File Lain

index.php 
<?php
// Memanggil file koneksi
include 'koneksi.php';

// Sekarang bisa pakai variabel $conn
echo "Sudah terkoneksi ke database!";
?>
