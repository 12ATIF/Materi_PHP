üõ°Ô∏è Bagian 6: Penjelasan Prepared Statement

üìñ Tipe Parameter bind_param:

Kode Tipe Data Contoh
s String (teks) nama, email, alamat
i Integer (angka bulat) id, umur, jumlah
d Double (desimal) harga, nilai
b Blob (binary) file, gambar
Contoh: mysqli_stmt_bind_param($stmt, "ssi", $nama, $email, $id);
Artinya: 2 string + 1 integer

‚ùå Query Biasa (RENTAN!)

$sql = "DELETE FROM siswa WHERE id = $id";

Bisa dibobol jika $id berisi: 1 OR 1=1

‚úÖ Prepared Statement (AMAN!)

$stmt = mysqli_prepare($conn, "DELETE FROM siswa WHERE id = ?");

Input apapun akan di-escape otomatis

  2. Tambahkan fitur pencarian berdasarkan judul buku
  3. Tambahkan pagination (10 data per halaman)

File-File CRUD:

  • ‚Ä¢ koneksi.php - Koneksi database

Best Practices:

  • ‚úÖ Selalu pakai Prepared Statement
  • ‚úÖ Cast ID ke integer: (int) $_GET['id']

  • ‚úÖ Redirect setelah proses berhasil
